# Crimes contre les biens

Synthèse — vol, escroquerie, abus de confiance, destruction, recel, infractions numériques.

## Objectifs

- Qualifier le **moyen** (rupture, ruse, abus…) et le **but** (appropriation, destruction…).
- Maîtriser les **concours** et la **complicité**.

## Tableau rapide

| Famille | Indices dans les faits |
|---------|-------------------------|
| Vol | Soustraction **sans** consentement |
| Escroquerie | **Ruse** déterminante |
| Abus de confiance | Détention **légitime** puis détournement |
| Recel | Connaissance de l’origine illicite |
